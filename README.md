# NeuroRad Reporter

**AI-powered neuroradiology structured report generator**

🌐 **Live at: https://Radiology4U.github.io/NeuroRadReporter/**

## Features

- 🖥️ **DICOM viewer** — drag & drop files, scroll slices, window/level, zoom, 5 presets
- 📄 **Dual-format reports** — long narrative + structured view with ICD-10 codes
- 💬 **AI Chat** — multi-turn conversation grounded in the active report
- ⬡ **Bidirectional PACS** — Orthanc, HL7 FHIR R4, DICOMweb, DICOM SR, Clipboard

## Usage

Open the app, enter your Anthropic API key, input findings, generate report.

> ⚠️ For educational and assistance purposes only. Not for clinical use without expert radiologist review.
